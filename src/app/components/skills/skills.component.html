Habilidades

Ao longo da minha jornada como desenvolvedor tive contato com diversas tecnologias e até hoje estudo novas tecnologias para complementar minhas habilidades, algumas delas são:

    Utilizo a linguagem C# juntamente com o framework .NET seguindo padrões de orientação a objetos e padrões de arquitetura para construir módulos de backend juntamente com ORMs e queries para acesso a banco de dados.
    Juntamente com C#, utilizo o framework .NET para desenvolver APIs robustas para o backend das aplicações, utilizando tecnologias de acesso a banco de dados, autenticação e autorização de usuários, documentação de APIs entre outras coisas.
    Com a linguagem TypeScript juntamente com os frameworks Angular e React desenvolvo o frontend das aplicações, extraindo o poder da tipagem para melhorar a qualidade e manutenibilidade do código.
    Utilizo o framework Angular para organizar e facilitar o desenvolvimento do frontend das aplicações, seguindo um padrão orientado a componentes, módulos, models e services, tornando o desenvolvimento das aplicações mais organizado e fácil de dar manutenção.
    Utilizo o framework React no desenvolvimento de aplicações robustas SPA, utilizando bibliotecas diversas que complementam o desenvolvimento com funcionalidades úteis, como Axios para realizar requisições HTTP Rest, React Router para gerenciamento de rotas, Clerk para autenticação de usuários, hooks diversos para manipulação de estado, eventos, contextos e etc.
    Utilizo o Tailwind para estilização descomplicada nos projetos Frontend, utilizando uma estilização orientada a classes e extensões pontuais quando necessário, como cores adicionais, tipografia ou espaçamento que o Tailwind pode não ter nativamente.
    Com o banco de dados SQL Server posso manipular os dados que serão utilizados nas aplicações através de ORMs e Queries escritas no backend.
    Cheguei a ter contato e utilizar o Azure Devops para entrega contínua (CI/CD) do projeto, versionando o código através do Git.
    Essa plataforma também serviu para organização e planejamento do time que trabalhei junto desenvolvendo e escrevendo estórias de usuário, seguindo a metodologia ágil SCRUM.
