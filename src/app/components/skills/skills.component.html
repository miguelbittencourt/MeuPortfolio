Habilidades

Durante o período de estágio tive contato com um ambiente de desenvolvimento com as seguintes tecnologias:

    Utilizei a linguagem C# juntamente com o framework .NET seguindo padrões de orientação a objetos e arquitetura DDD para construir módulos de backend juntamente com ORMs e queries para acesso a banco de dados, além disso, escrevia testes unitários para garantir a qualidade do código e da aplicação.
    Juntamente com C#, utilizei o framework .NET para desenvolver APIs robustas para o backend da aplicação interna da empresa que estagiei, utilizando tecnologias de acesso a banco de dados, autenticação e autorização de usuários, documentação de APIs entre outras coisas.
    Com a linguagem TypeScript juntamente com o framework Angular desenvolvi o frontend dos módulos internos, utilizando a biblioteca do Angular Material para facilitar a estilização do layout das aplicações, também escrevi testes unitários para garantir a qualidade do código.
    Utilizei o framework Angular para organizar e facilitar o desenvolvimento do frontend das aplicações, seguindo um padrão orientado a componentes, módulos, models e services, tornando o desenvolvimento das aplicações mais organizado e fácil de dar manutenção.
    Com o banco de dados SQL Server pude manipular os dados que seriam utilizados nas aplicações através de ORMs e Queries escritas no backend.
    Utilizei o Azure Devops para entrega contínua (CI/CD) do projeto, versionando o código através do Git.
    Essa plataforma também serviu para organização e planejamento do time que trabalhei junto desenvolvendo e escrevendo estórias de usuário, seguindo a metodologia ágil SCRUM.
